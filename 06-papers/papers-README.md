# 06 · Paper Reading Log

2+ papers per week. Every paper gets its own structured note.

---

## Note Format

Each file covers:
- **Problem** — what are they solving
- **Key idea** — the core contribution in one sentence
- **The math** — key equations worked through properly
- **Connection to my work** — how it relates to what I've built
- **Open questions** — what I still don't fully understand
- **Next steps** — what to read or implement next
